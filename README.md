# seasoning-management-iOS
調味料を管理するiOSアプリ

## 開発環境
- Xcode 11
- swift 5
- CocoaPods 1.8.4

## 環境構築
1. ソースコードダウンロード
2. 以下のコマンドを実行
```
$ pod install
```
3. 生成されたxcworkspaceファイルを開く
